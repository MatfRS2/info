#### Формулар за пријављивање пројеката

**Датум објаве: 25.11.2024.**

Тему за пројекат можете званично пријавити путем [овог формулара](https://forms.gle/r57J4PnV66cGzk6e7). Тимови који су прошле године пријавили пројекат и желе да наставе са радом потребно је да поново пријаве пројекат. Рок за пријављивање тема је 25. децембар. Сви студенти који не буду распоређени у неки тим до тада биће распоређени у тимове који ће бити формиране од стране асистента (случајним одабиром).
