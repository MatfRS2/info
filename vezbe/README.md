# Вежбе -- Развој софтвера 2 @ Математички факултет Универзитета у Београду

[РС 2](../README.md)

Овде је описан садржај вежби за предмет **Развој софтвера 2**. У оквиру овог курса се проучава развој микросервисних апликација. Користи се програмски језик C# и развојни оквир .NET 9 заједно са одговарајућим библиотекама.

## Садржај вежби

1. Елементарно о микросервисима у развојном оквиру .NET 10
    - Теме:
        - Микросервиси
        - .NET 10 развојни оквир
        - Окружења за развој, алат `dotnet`
        - Преглед концепата језика C#
    - [Поглавље о микросервисима из скрипте у припреми.](./ukratko-o-mikroservisima.pdf)
    - [Белешке](./beleske.pdf#page=3)
	- [Помоћ за инсталацију ресурса](./pomoc-za-instalaciju.pdf)

1. Развој микросервиса уз развојни оквир .NET 10
    - Теме:
        - Контејнеризација микросервиса и екстерних ресурса: Docker
        - JetBrains Rider; Web API пројекти
        - Повезивање на MongoDB СУБП
    - Микросервиси:
        - Catalog API
    - [Белешке](./beleske.pdf#page=3)
        - [Белешке са часа](./beleske/01_docker_catalog_api.md){:target="_blank"}
        - [Кратак туторијал за алат Docker](https://github.com/Robotmurlock/MATF-AZRS/tree/main/tema07_docker){:target="_blank"}
